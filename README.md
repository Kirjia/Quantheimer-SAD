## Dataset
The dataset is taken from a [polish study on alzheimer](https://openneuro.org/datasets/ds004796/versions/1.1.0).
The sample has 192 entries each of those provides features about gene's risk, psychometric (memory, intelligence, mood, personality, stress coping strategies), basic demographic and health data.
A subset of this dataset (79 samples) provide the blood test.
